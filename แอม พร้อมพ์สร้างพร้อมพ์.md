3.ระบบเฉพาะงานนี้
    3.1 ระบบนี้ชื่อ ระบบรับนักเรียนความสามารถพิเศษ talentStd
    3.2 แอดมิน ระบบ เข้าระบบด้วยรหัสผ่านตัวเลข 8 ตัว โดยเก็บใน ชีท config
        3.2.1 แอดมิน crud ชื่อการรับนักเรียนความสามารถได้ เช่น   คอมพิวเตอร์ ม.ต้น   / คอมพิวเตอร์ ม.ปลาย  / พละ ม.ต้น /พละ ม.ปลาย
        3.2.2 แอดมินของระบบ สามารถ CRUD กรรมการตัดสิน มีชื่อ สกุล กรรมการ และตำแหน่ง พร้อมมอบรหัสเป็นตัวเลข 8 ตัว ให้ กรรมการแต่ละคนสำหรับ login ได้
        3.2.3 แอดมินของระบบ สามารถ CRUD รายชื่อนักเรียน
        3.2.4 แอดมินของระบบ สามารถ CRUD ลักษณะความสามารถ เช่น ด้านคอมพิวเตอร์ / ด้านหุ่นยนต์ เพิ่มได้ไม่จำกัด
            3.2.3.1  แต่ละด้าน จะมีเกณฑ์ และแต่ละเกณฑ์มีหัวข้อย่อย + คะแนนเต็ม , แอดมิน เพิ่มลด เกณฑ์+หัวข้อ คะแนนเต็ม ได้   ตัวอย่าง เกณฑ์ทักษะการใชใ้เครื่องมือโปรแกรมที่นักเรียนถนัด หัวข้อย่อยเกณฑ์นี้ 1.การติดตั้งโประแกรม มีคะแนนเต็ม 10 คะแนน   2. ความถนัดความชำนาญในการใช้เครื่องมือ 20 คะแนนเต็ม เป็นต้น
          
    3.3 การเข้าถึงของกรรมการ สามารถเข้าถึงได้ง่ายด้วย qr code แล้ว ล๊อคอินด้วย รหัสที่แอดมินมอบให้ 8 ตัว (ระบบต้องสร้าง qr code ให้ กรรมการด้วยลิงค์เดียว)  กรรมการ มีหน้าที่กรอกคะแนน เข้าไปในแต่ละหัวข้อ ตามคะแนนเต็มสูงสุดที่มอบให้
    3.4 แอดมิน สามารถเช็กการรายงานของกรรมการได้ว่า รายงานครบหรือไม่ทุกคน คือต้องกรอกครบทุกคน แล้วสามารถส่งออกรายงานเป็น a4 แนวตั้ง เพื่อสรุปรายงาน และเว้นที่ให้กรรมการเซ็น  โดยในเอกสารนี้ให้แสดง หัวข้อในข้อ 3.2.1 ตัวใหญ่จัดกลางด้านบน  และมีตารางคะแนน เกณฑ์แต่ละด้าน พร้อมห้วข้อย่อย แสดงคะแนนเต็ม คะแนนเฉลี่ย(ของกรรมการทุกคน)  จากนั้นด้านล่างเป็นคะแนนรวม  ล่างสุดเว้นที่ให้กรรมการเซ็นชื่อ 1 แถว มีได้ 2 คอลัมพ์ สำหรับกรรมการ  มีตำแหน่ง และวันทีเว้นไว้ด้วย
    3.5 ระบบ logout ทั้งแอดมินและกรรมการ — ทุกหน้าหลัง login ให้มีปุ่ม "ออกจากระบบ" แสดงอยู่บน Header ขวามือ เมื่อกดจะล้าง session แล้ว redirect กลับหน้า login
